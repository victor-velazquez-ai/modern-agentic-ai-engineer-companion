# Security Onboarding and Acceptable Use

Every employee completes security onboarding within their first week. You must enable
multi-factor authentication (MFA) on your company account before accessing any internal system.
Hardware security keys are provided on request and are required for anyone with production access.

Never reuse your company password on external sites. Use the company password manager for all
work credentials. Report any suspected phishing email to security@company.example using the
"Report Phishing" button; do not forward it to colleagues.

Company data is classified as Public, Internal, Confidential, or Restricted. Restricted data
(such as compensation, unreleased financials, and personal employee records) may only be
accessed by people with an explicit business need and the corresponding access group.

Lost or stolen devices must be reported to IT within one hour so the device can be remotely
wiped. Do not store Restricted data on personal devices under any circumstances.
