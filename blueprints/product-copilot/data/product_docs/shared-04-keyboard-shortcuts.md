# Keyboard shortcuts

Press **?** anywhere in Nimbus to see the full shortcut list. The essentials: **Cmd/Ctrl+K**
opens the command palette, **G then P** jumps to Projects, **G then D** opens your Dashboard, and
**Cmd/Ctrl+Enter** runs the current query. Shortcuts speed up everything once they are muscle
memory, so the command palette is the fastest way to navigate.
