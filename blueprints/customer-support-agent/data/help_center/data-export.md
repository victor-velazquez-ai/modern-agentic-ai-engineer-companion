# Exporting your data

You own your data and can export it at any time. Go to Settings → Data → Export to generate a
download. We package your account's content as JSON and CSV and email you a secure, expiring link
when the export is ready, usually within a few minutes for most accounts.

Exports do not include other workspaces' data or another member's private content. If you need a
machine-readable export for a migration, the API also exposes the same data under /v1/export.
