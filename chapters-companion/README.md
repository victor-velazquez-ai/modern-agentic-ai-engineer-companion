# Learn — chapter notebooks

The 📓 pillar: one folder per book chapter, mirroring the book's 13 parts and 54 chapters.
Each chapter folder has a `PLAN.md` listing its planned notebooks — what each teaches, which
book sections it maps to, and which blueprint/template/capstone it feeds.

> **Phase 1:** these are plans, not notebooks yet. See the repo [README](../README.md) and
> [docs/REPO-PLAN.md](../docs/REPO-PLAN.md) for the build phases, and
> [docs/HOW-TO-USE.md](../docs/HOW-TO-USE.md) for reading paths.
